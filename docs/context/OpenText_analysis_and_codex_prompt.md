# Анализ архива OpenText / Matrix

## Что разобрано
- Matrix.zip — репозиторий userscript-проекта, фикстуры HTML, тесты, AGENTS, README, пресеты, historical artifacts.
- HelpDesk_Export_20260427_124819.zip — архив MSG по обращениям OpenText.
- Матрицы.zip — выгрузки текущих матриц по договорам/доверенностям/ПВП/инвестпроектам.
- Своды.zip — своды полномочий, подписанты, спецэксперты, презентации.

## Короткий вывод
Проект должен быть не одним compact userscript, а тремя связанными контурами:
1. Matrix Operator — массовые изменения матриц.
2. Card/Route Doctor — диагностика карточки и причин, почему не строится ЛС.
3. HD/ITCM Triage — классификация обращений и полуавтомат для первой линии.
