# Руководство по вкладу

## Быстрый старт

1. Установите зависимости:
   - `npm install`
2. Соберите userscript:
   - `npm run build`
3. Запустите проверки:
   - `npm run test:unit`
   - `npm test`
   - `npm run smoke`

## Правила изменений

- Любая массовая мутация должна иметь визуальное превью.
- Для операций с контрагентами обязательна аффилированность `Группа Черкизово`.
- Любая новая операция должна возвращать понятный `reason`/`sourceRule`.
- Не ломайте совместимость API без обновления документации и migration guide.

## Перед Pull Request

- Обновите документацию, если меняется поведение.
- Добавьте/обновите тесты.
- Убедитесь, что рабочее дерево чистое кроме релевантных файлов.
